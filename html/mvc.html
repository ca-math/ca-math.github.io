Problem 1

a) With the aid of a drawing, stretch or shrink vectors a and b so that their sum, using the parallelogram rule, gives the vector c.


b) For an algebraic argument, set up the equations:

a = <a1, a2>

b = <b1, b2>

c = <c1, c2>


Finding t and s amounts to solving the system of equations:

c1 = s a1 + t b1

c2 = s a2 + t b2


Because the vectors are nonzero and not parallel, this system will have a unique solution.
