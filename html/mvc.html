Problem 1

a) With the aid of a drawing, stretch or shrink vectors a and b so that their sum, using the parallelogram rule, gives the vector c.


b) For an algebraic argument, set up the equations:

a = <a1, a2>

b = <b1, b2>

c = <c1, c2>


Finding t and s amounts to solving the system of equations:

c1 = s a1 + t b1

c2 = s a2 + t b2


Because the vectors are nonzero and not parallel, this system will have a unique solution.


Problem 2

Parts a through c, require drawing.

d) Set up the equations, similar to problem 1, to find s=9/7 and t=11/7.


Problem 3

After drawing neatly, write

c = a + AC

c = b - 2AC 

Multiply the first equation by 2, add side by side, and then solve for c.


Problem 4

Let v = <a , a, a>

Determine that sqrt(3) a = 1, so that a = sqrt(3)/3 and then list the component form.
